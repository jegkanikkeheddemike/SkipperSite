Directtransfer

Directransfer er både en CLI og en desktop app som kan bruges til at overføre filer over LAN forbindelser.
Kildekoden kan findes på https://github.com/jegkanikkeheddemike/ThorProjekter

Download

Directransfer kan sættes til to "modes": HOST og LEECH. Host bruges hvis man har en fil man vil dele. og Leech bruges til at downloade en fil som en anden hoster.
I CLI'en skriver man derfor:
directtransfer.exe host "C:/min_fil_eller_mappe"
eller
directtransfer.exe leech "hosterns ip:port"
Bemærk at det er lavet til at overføre med LAN forbindelser. Men hvis man åbner en port i sit internet kan man også overføre gennem den

Normalt bliver filerne overført til C:/ProgramData/directtransfer/output men man kan enten skrive directtransfer.exe settings set output_folder "C:/min_nye_output_mappe"
eller
directtransfer.exe leech "ip:port" "C:/min_midlertidige_output_mappe"
Man kan tjekke sine settings ved at skrive
directtransfer.exe settings get
